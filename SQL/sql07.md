# PARTITION
- 데이터 정리 용도
- 장점
  1. 쿼리 성능 향상(파티션 설정해 스캔 범위 줄임)
  2. 데이터 관리 용이성(특정 일자 데이터 모두 변경시 파티션 활용)
  3. 비용(파티션 해당 데이터만 스캔해서 비용 줄일 수 있음)

# 데이터 검증

- 방법 : 1. 기대하는 예상 결과 정의     
        2. 쿼리 작성
        3. 1-2 일치하는지 확인
